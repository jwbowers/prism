How it works

From zero to running in three steps

No AWS expertise required.

Configure your profile

Connect your AWS credentials once with prism profile add. Prism handles the rest.

Choose a template

Pick from built-in research environments or browse the community marketplace.

Launch and connect

Run prism workspace launch python-ml my-project. SSH access is ready in under two minutes.

Stop when done

Hibernate to pause billing while keeping your data. Terminate to clean up completely.

Installation

Get up and running

Prism is a single binary. Install it, connect your AWS credentials, and launch.

# Install via Homebrew
brew tap scttfrdmn/prism
brew install prism

# Or download the GUI desktop app below

# First-time setup
prism profile add
prism workspace launch python-ml my-first-project
# Install via Homebrew on Linux
brew tap scttfrdmn/prism
brew install prism

# Or download the binary directly (see Downloads below)
tar -xzf prism-linux-amd64.tar.gz
mv prism /usr/local/bin/

# First-time setup
prism profile add
prism workspace launch python-ml my-first-project
# Install via Scoop
scoop bucket add scttfrdmn https://github.com/scttfrdmn/scoop-bucket
scoop install prism

# Or download the GUI desktop app below

# First-time setup
prism profile add
prism workspace launch python-ml my-first-project

Requires AWS credentials with EC2 access. See the IAM permissions guide for the minimal policy.
